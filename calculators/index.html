근로기준법 상 근로자 인건비 계산기 (인건비 기본급 및 주휴수당 예산 구분)
메인으로 돌아가기

주 소정 근로시간이 15시간 이상일 경우 근로기준법에 따라 주휴수당 부여해야 함.

엑셀 파일로 저장
구분 주중 기본급 합계 (원단위) 주휴수당 합계 (원단위) 총 금액 (원단위)
금액 0 0 0
예산 대비 잔액 0 0 0

예산 잔액 초과됨

산식
주간 유급시간 = weeklyHours + (volPaid==="Y" ? volHours : 0)
주휴 인정 여부 = (주간 유급시간 ≥ 15)
주당 주휴시간 = 주휴 인정 시 (주간 유급시간 ÷ weeklyDays) , 아니면 0
주당 주중임금 = 주간 유급시간 × 시급
주당 주휴수당 = 주당 주휴시간 × 시급
전체 합계 = (주중임금 × 총 주 수) + (주휴수당 × 총 주 수)
